Search conditions

Adding as simple condition to get pets, at least 1 year old

Path: %path%?query={"age":{"$ge":1}} 
{
    "searchMechanism": "search",
    "conditions": {
        "age": {
            "$ge": 1
        }
    },
    "docs": [{
        "name": "Pupper",
        "age": 1,
        "_id": "1",
        "_created": "2018-03-16T17:26:56.169Z",
        "_updated": "2018-03-16T17:26:56.169Z"
    }]
}
